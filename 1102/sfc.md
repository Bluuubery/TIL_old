## Component in Vue

✔ **이름이 있는 재사용 가능한 Vue instance**  
✔ `new Vue()`

## SFC(Single File Component)

✔ 하나의 `.vue`파일이 하나의 Vue instace이고, 하나의 컴포넌트!  
✔ Vue instance에서는 HTMl, CSS, JavaScript 코드를 한번에 관리
