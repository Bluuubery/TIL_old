# SQL: Intro
## SQl이란
✔ Structured Query Language  
✔ **RDBMS의 데이터를 관리하기 위해 설계된 특수 목적**의 프로그래밍 언어  
✔ **데이터와 상호작용하기 위한 언어**  
✔ RDBMS에서 데이터베이스 스키마를 생성 및 수정할 수 있으며, 테이블에서의 자료 검색 및 관리도 할 수 있다
